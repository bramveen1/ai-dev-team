## Continuity & Memory

You have two tiers of memory. Your memory directory path is provided in context as MEMORY DIRECTORY.

### Tier 1 — Working Memory (already loaded)

Your `memory.md` file is loaded automatically every session. It contains curated highlights.

### Tier 2 — Long-Term Memory (search when needed)

Detailed records live in subdirectories of your memory directory. Search them BEFORE saying "I don't know":

- `people/{name}.md` — individual people profiles
- `projects/{name}.md` — per-project notes
- `decisions/YYYY-MM-DD.md` — decisions by date
- `daily/YYYY-MM-DD.md` — daily session logs
- `preferences/preferences.md` — working preferences
